# AI UGC Ad Content Generator

## 📌 Project Overview
This project demonstrates how Prompt Engineering can be used to create high-converting UGC (User Generated Content) ad scripts for modern digital marketing.

The goal was to design a reusable prompt system that generates:
- Scroll-stopping hooks
- UGC-style ad scripts
- Call-to-actions (CTAs)
- Instagram captions

These outputs are designed for platforms like Instagram Reels and YouTube Shorts.

---

## 🏷️ Product Details
- **Product:** SonicBeat X1 Headphones  
- **Type:** Wireless Noise-Cancelling Headphones  
- **Target Audience:** Students, remote workers, gamers  

### Problem Solved:
- Background noise distractions  
- Lack of focus during study/work  
- Poor audio experience from cheap headphones  

---

## 🧠 Prompt Engineering Approach

### 1. Role-Based Prompting
The AI is guided to act as a UGC creator or ad scriptwriter to produce realistic content.

### 2. Structured Framework
All scripts follow a proven format:
Hook → Problem → Solution → Proof → CTA

### 3. Content Variation
Prompts are designed to generate:
- Multiple hooks
- Different script styles
- Diverse CTAs

### 4. Platform Optimization
Content is optimized for short-form platforms like Instagram and YouTube Shorts.
